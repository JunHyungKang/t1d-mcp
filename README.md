# 🩸 T1D Manager (1형 당뇨 관리 MCP)

**카카오 MCP Player 10 공모전 출품작**

1형 당뇨 환자와 보호자를 위한 AI 보조 에이전트 서버입니다.
실시간 혈당 모니터링(Dexcom/Nightscout), 정밀한 인슐린 계산, 그리고 따뜻한 멘탈 케어를 제공합니다.

## ✨ 주요 기능

1.  **실시간 혈당 브리핑**: "지금 혈당 어때?" -> Dexcom 데이터를 Nightscout에서 가져와 분석.
2.  **인슐린 교육 계산기**: "혈당 200인데 밥 먹을 거야" -> 기저/초속 인슐린의 원리를 그림(표)과 함께 설명하며 용량 계산.
3.  **안심 영양 검색**: "순대국밥 탄수화물 몇이야?" -> 검증된 영양 정보 제공.
4.  **커뮤니티 지식 검색**: 네이버 카페/블로그와 다음 검색을 통합하여 환자들의 생생한 꿀팁 제공.

## 🚀 시작하기

### 1. 환경 설정
Python 패키지 매니저 `uv`가 필요합니다.

```bash
# 프로젝트 클론
git clone https://github.com/JunHyungKang/t1d-mcp.git
cd t1d-mcp

# 의존성 설치
uv sync
```

### 2. 환경 변수 설정
`.env.example`을 복사하여 `.env`를 생성하고 키를 입력하세요.

```bash
cp .env.example .env
```

### 3. 실행 (MCP Server)

```bash
uv run src/main.py
```
