# MCP 가이드라인

이 문서는 MCP 서버 생성 및 심사 정책에 관한 가이드라인을 담고 있습니다.

## MCP 서버 생성 가이드
- **링크**: [MCP 서버 생성 가이드](https://www.notion.so/MCP-2d89b97b4888808a9e1dc17a13e70187)
- **주요 내용**: MCP 개발 및 제출 시 참고해야 할 서버 생성 가이드

## MCP 심사 정책
- **링크**: [MCP 심사 정책](https://www.notion.so/MCP-21b9b97b48888024922ec3dfcacf97e5)
- **주요 내용**: MCP PLAYER 10 공모전 관련 심사 기준

## 유의 사항
- 가이드 기준에 맞지 않을 경우 심사 과정에서 반려될 수 있으니 반드시 확인이 필요합니다.
- 공모전 응모자는 심사 완료 후 MCP 상태를 **전체 공개**로 설정해야 합니다.
- 문의 사항은 질의응답 채널 또는 1:1 문의 채널을 활용하세요.
